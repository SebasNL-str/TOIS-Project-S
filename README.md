# TOIS-Project-S
This is an immersive 3D scene depicting a small cemetery on a night with a full moon. The focus is not on the size of the map, but on visual coherence. The user can walk along a path flanked by old gravestones, a wrought-iron fence, and trees, while the faint glow of lanterns flickers through the fog.
this is a game for the creator of Sizophreni.
